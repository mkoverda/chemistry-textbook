# Учебник по общей химии

Это небольшой проект, посвящённый созданию учебника по общей химии в авторском изложении. Учебник рассчитан на начинающих и создаётся при помощи LaTeX. Учебник пока неполный.

## Сборка

Просто скачайте архив с одним из релизов или файлами проекта и соберите документ:

```bash
pdflatex main.tex
```

Обратите внимание, что в рабочей версии могут быть проблемы с форматированием и недоработанные разделы.
